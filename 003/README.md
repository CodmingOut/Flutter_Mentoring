# Hello World!

다트 언어에서 'Hello World!' 를 출력해봅시다.

<br>

## 메인 함수

```dart
void main() {
	print('Hello World!');
}
```

```
Hello World!
```

자바의 main 메서드나 C언어의 main 함수처럼, 다트 언어도 main 함수를 가지고 있습니다.

다른 여느 언어처럼, 다트 언어도 main 함수에서 프로그램이 시작됩니다.

다트 언어에서 값 출력은 `print();` 함수를 통해 할 수 있습니다.
